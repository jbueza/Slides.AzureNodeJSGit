In This Talk

I'll be going over a few awesome features that landed recently...

  • Git-based deployment: Adds support for deploying web apps with Git
  • Apps: Added "ready to go" applications, such as, Wordpress, Drupal, DotNetNuke. Also support for MySQL databases.
  • SDKs: Added support for NodeJS, PHP, Python, and Java

Each extension folder in the download package contains the necessary JavaScript, CSS, and HTML files. For a complete list of extension modules included in deck.js, check out the documentation.
